## Overview

### The challenge

Users should be able to:

- View the optimal layout for the app depending on their device's screen size
- See hover states for all interactive elements on the page
- Add new todos to the list
- Mark todos as complete
- Delete todos from the list
- Filter by all/active/complete todos
- Clear all completed todos
- Toggle light and dark mode
- **Bonus**: Drag and drop to reorder items on the list

### What I learned

In this project, I applied what I learned from a React course by adopting a component-based programming style. I used hooks, context, and custom hooks, as well as localStorage for data persistence, and third-party libraries such as ShadCN, Tailwind, and dnd-kit.

I also learned how to use Tailwind effectively, leveraging its utility classes to style HTML elements more efficiently. Compared to writing custom CSS classes, it allows for faster and more streamlined development.
